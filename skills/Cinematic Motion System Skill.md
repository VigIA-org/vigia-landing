# Motion Principles

## 1. Smooth Scrolling

Implement buttery smooth scrolling.

Requirements:
- inertia-based scroll
- interpolation
- velocity smoothing
- frame-synced scrolling

Preferred stack:
- Lenis
- GSAP ticker integration

The scroll should feel:
- weighted
- cinematic
- fluid
- responsive

Never use raw browser scrolling alone for premium pages.

---

## 2. GPU-Accelerated Motion Only

Animate ONLY:
- transform
- opacity
- filter

Avoid animating:
- width
- height
- top
- left
- margin
- padding

Prefer:
- translate3d
- scale
- rotate
- opacity
- blur

Always use:
- will-change
- transform-gpu

Performance is mandatory.

---

## 3. Reveal Animations

Elements should reveal progressively.

Every reveal should combine:
- opacity
- translation
- slight scale
- blur reduction

Example behavior:

Initial:
- opacity: 0
- y: 40px
- scale: 0.98
- blur: 10px

Final:
- opacity: 1
- y: 0
- scale: 1
- blur: 0px

Timing:
- slow
- elegant
- cinematic

Never use instant fade-ins.

---

## 4. Stagger Systems

Never animate groups simultaneously.

Use staggered choreography for:
- headings
- paragraphs
- cards
- buttons
- navigation
- feature grids

Recommended stagger:
- 0.04s → 0.12s

Motion should feel orchestrated.

---
